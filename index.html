Configure Outlook Manually

To configure your email accounts manually with Microsoft Outlook, follow these steps:

Step 1: Open Microsoft Outlook on your computer. Click File > Account Settings > Manage Profiles.
Step 2: In the Mail Setup - Outlook window, click Show Profiles, select your profile, and click Properties. Then click Email Accounts.
Step 3: In the Account Settings window, click New to add a new email account.
Step 4: Select Manual setup or additional server types and click Next.
Step 5: Choose POP or IMAP and click Next.
Step 6: Fill in the following details:
  • Your Name: Enter the name you want to appear in outgoing emails.
  • Email Address: Enter your full email address (e.g., username@yourdomain.com).
  • Account Type: Select IMAP (recommended, unless you specifically need POP3).
  • Incoming mail server: Find the exact hostname in Site Tools > Email > Accounts, select the email account, go to the kebab menu > Mail Configuration, and check the Manual Settings tab.
  • Outgoing mail server (SMTP): Use the same hostname as the incoming server.
  • User Name: Enter your full email address (e.g., username@yourdomain.com).
  • Password: Enter the password for your email account.
Step 7: Click More Settings, then go to the Outgoing Server tab and check My outgoing server (SMTP) requires authentication. Select Use same settings as my incoming mail server.
Step 8: In the Advanced tab, set the following:
  • Incoming server (IMAP): Port 993, Encryption method: SSL/TLS.
  • Outgoing server (SMTP): Port 465, Encryption method: SSL/TLS.
Step 9: Click OK, then Next. Outlook will test the account settings. If successful, click Finish to complete the setup.
Note: To verify settings, go to Site Tools > Email > Accounts, select the email account, and check Mail Configuration > Manual Settings tab.

Edit Existing Email Account Settings in Outlook

Grok 3 is unable to find instructions to edit an existing email account in Microsoft Outlook. Instead, try the following:

Alternative: Delete the existing email account and re-add it with the updated settings.
  • Go to File > Account Settings > Account Settings.
  • Select the account you want to edit and click Remove.
  • Follow the steps in the "Configure Outlook Manually" section above to re-add the account with the correct settings.
Note: Ensure you have the correct settings from Site Tools > Email > Accounts > Mail Configuration > Manual Settings tab before making changes.
